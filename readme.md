# Plataforma de Recrutamento

## About
Uma plataforma para envio e gerenciamento de currículos.

### Tecnologias
  * Laravel 5.6
  * PHP 7.1
  * Material Design Lite
  * MongoDB

## Instalação
> Configure o seu .env
> Entre no diretório do projeto e execute os seguintes comandos
```
  - $ npm install
  - $ composer install
  - $ yarn run dev
  - $ yarn watch
```
